# 🚦 Crash Risk Intelligence System

End-to-end data science project analyzing 215,000+ real Maryland 
police crash records to predict crash risk and severity.

## Project Structure
- `notebooks/` - Jupyter notebooks for cleaning, EDA, feature engineering, and modeling
- `data/` - Raw dataset (not tracked by Git)
- `models/` - Saved ML model files

## Notebooks
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. ML Modeling

## Tech Stack
Python, Pandas, Scikit-learn, CatBoost, XGBoost, Matplotlib, Seaborn

## Data Source
[Maryland Crash Reporting - Drivers Data](https://catalog.data.gov/dataset/crash-reporting-drivers-data)
